## 开发说明

  本程序是用python3编写， 少量前端用js编写。  

### 项目依赖

#### python3

* wxpython GUI支持

*requests http web支持

*pywin32 SAPI支持

*pygame 音效支持

* pyinstaller 支持python程序打包生成exe文件

### 文件说明

* app.py 应用程序入口

* game.py 核心游戏实现

* manager.py 游戏管理模块， 包括游戏交互； 用户界面； web交互等等

* tts.py 语音接口实现

* ssetting.json 默认服务器端配置

* py2exe.spec 生成exe文件的pyinstaller脚本

  后端代码请参看service文件夹。
